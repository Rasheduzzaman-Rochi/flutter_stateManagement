# Flutter State Management Practice 🚀

This repository contains small Flutter projects demonstrating different state management techniques.

## 📌 Purpose

The goal of this repository is to understand and compare various state management approaches in Flutter by building small practical projects.

---

## 🧠 Covered State Management Techniques

### 1️⃣ setState
- Counter App
- Theme Switcher

### 2️⃣ Provider
- Shopping Cart App
- Todo App

### 3️⃣ Riverpod
- API Fetch Example
- Todo with StateNotifier

### 4️⃣ BLoC
- Counter with Bloc
- Login Form Validation

### 5️⃣ GetX
- Simple Login App
- Cart Example
